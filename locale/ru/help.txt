Использование:
  gitk <команда> [опции]

Доступные команды:
  init        Инициализирует новый git-репозиторий
  add         Добавляет указанные файлы в индекс
  commit      Генерирует сообщение коммита с помощью нейросети
  push        Отправляет изменения на удалённый репозиторий
  help        Показывает эту справку

Поддерживаются также обычные команды git, если нет своей реализации.

[init]
Описание:
  Инициализирует новый git-репозиторий в текущей директории.

[add]
Описание:
  Добавляет указанные файлы в индекс.

Примеры:
  gitk add file1.txt file2.py
  gitk add .         # добавляет всё

После добавления будет показан список файлов, попавших в индекс.

[commit]
Описание:
  Генерирует сообщение коммита на основе изменений в индексе,
  используя нейросетевой генератор.

Опции:
  --detailed         Генерация подробного сообщения коммита
  --yes, --no-confirm  Не спрашивать подтверждение перед коммитом
  --author, --amend и другие опции Git также поддерживаются

[push]
Описание:
  Отправляет текущую ветку на удалённый репозиторий.

Особенности:
  - Если не указаны remote и ветка, будет предложен выбор
  - Используется remote по умолчанию: origin

Примеры:
  gitk push
  gitk push origin dev

[help]
Описание:
  Показывает эту справку. Можно указать команду:

Примеры:
  gitk help
  gitk help commit
